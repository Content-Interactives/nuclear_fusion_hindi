# नाभिकीय संलयन इंटरैक्टिव — तकनीकी README

यह **पाँच-चरणीय**, **500×500 px** निर्देशित अनुभव सूर्य में नाभिकीय संलयन पर आधारित है: पाठ-आधारित दृश्य, सूर्य के आंतरिक भाग का दृश्य, चार **हाइड्रोजन** परमाणुओं को **संलयन क्षेत्र** में खींचकर छोड़ना, फिर **CSS एनीमेशन** से संलयन व ऊर्जा का प्रदर्शन। पूरा कार्यान्वयन **एक स्थिर HTML फ़ाइल** में है—कोई बंडलर या बाहरी निर्भरता नहीं। **उपयोगकर्ता-मुखी पाठ हिंदी में** है; स्क्रिप्ट में चरण बटन व काउंटर टेक्स्ट हिंदी है।

**लाइव बिल्ड:** [https://content-interactives.github.io/nuclear_fusion_hindi/](https://content-interactives.github.io/nuclear_fusion_hindi/)

पाठ्यक्रम, प्लेसमेंट और मानकों का विवरण [Standards.md](Standards.md) में है।

## DOM संरचना

- **`.container`** — निश्चित `500px × 500px`, `body` पर केन्द्रित, गहरा रेडियल पृष्ठभूमि।
- **`#stars`** — `createStars()` **50** `.star` div बनाता है (यादृच्छिक स्थिति, आकार, `animation-delay`)।
- **`.progress`** — पाँच `.progress-dot`; कक्षाएँ `active` / `completed` व **`currentStage`** से मेल खाती हैं।
- **स्टेज** — `#stage1` … `#stage5` वर्ग **`.stage`**; एक समय में केवल एक पर **`.active`** (`opacity: 1`, `pointer-events: auto`)।
- **`#navButton`** — नीचे केन्द्र; चरणानुसार हिंदी लेबल (जैसे `जारी रखें`, `और गहराई में जाएं`, `परिणाम देखें`, `फिर से शुरू करें`) व **`hidden`** क्लास।

### चरण मानचित्र

| `currentStage` | `#stageN` | भूमिका |
|----------------|-----------|--------|
| 1 | `stage1` | सूर्य + प्रश्न |
| 2 | `stage2` | भीतरी विवरण; `#sunInterior` पर `.visible` |
| 3 | `stage3` | चार हाइड्रोजन, `#fusionZone`, काउंटर, ड्रैग-ड्रॉप |
| 4 | `stage4` | फ्लैश, उत्सर्जित इलेक्ट्रॉन, हीलियम; `triggerFusion()` |
| 5 | `stage5` | छोटा सूर्य, ऊर्जा किरणें, समापन पाठ |

`goToStage(n)` सक्रिय स्टेज बदलता है, डॉट अपडेट करता है, व `switch` में `setTimeout`, बटन टेक्स्ट व `initDragAndDrop()` (केवल स्टेज 3) चलाता है।

## JavaScript व्यवहार

### वैश्विक स्थिति

- `currentStage` (1–5), `atomsInZone`, `hydrogenAtoms` (आइडी → `{ element, inZone, originalPos }`)
- ड्रैग: `isDragging`, `draggedAtom`, `dragOffset`

### पॉइंटर (स्टेज 3)

- `mousedown` / `touchstart` पर परमाणु; दस्तावेज़ पर `mousemove` / `touchmove` / `mouseup` / `touchend`।
- स्थिति `left`/`top` px में, **0–450** तक क्लैम्प।
- **संलयन क्षेत्र:** परमाणु केन्द्र व क्षेत्र केन्द्र के बीच यूक्लिडियन दूरी **&lt; 70** हो तो `inZone`, चार निश्चित **`zonePositions`** पर स्नैप। चारों ज़ोन में होने पर `#fusionZone` पर `.ready`, फिर **800 ms** बाद `goToStage(4)`।

### संलयन अनुक्रम (`triggerFusion`)

`setTimeout` श्रृंखला: फ्लैश + `.fly` → हीलियम `.visible` → नेव बटन (`परिणाम देखें`)।

### रीसेट

स्टेज 5 पर क्लिक: मूल स्थिति, कक्षाएँ साफ़, `goToStage(1)`।

## CSS नोट्स

- `@keyframes`: `twinkle`, `pulse`, `plasmaMove`, `orbit`, `zoneGlow`, `flash`, `flyOut1`–`4`, `rayPulse` आदि।
- **`.sun.zoomed`** CSS में परिभाषित है पर वर्तमान स्क्रिप्ट में **टॉगल नहीं** होता (भविष्य हुक या अवशिष्ट)।
- ऊर्जा **किरणें:** `createEnergyRays()` **16** `.ray` तत्व, `rotate` + विलंबित `animation-delay`।

## रखरखाव / विस्तार

- **हिट त्रिज्या** या **स्नैप जाल** बदलने पर स्क्रिप्ट में **`70`** व **`zonePositions`** के साथ CSS में `.fusion-zone` आकार समन्वयित रखें।
- नए चरण: नया `#stageN`, प्रोग्रेस डॉट, `goToStage`, नेव क्लिक हैंडलर व `reset()` में शाखाएँ।
- स्टेज 3 पर `initDragAndDrop()` एक बार चलता है; बिना पूर्ण रीलोड के स्टेज 3 दोबारा खोलने पर **डुप्लिकेट लिसनर** से बचाव आवश्यक हो सकता है।
